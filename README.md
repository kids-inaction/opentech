Εξερευνητές των αισθήσεων (1ο Νηπιαγωγείο Αριδαίας)

Εξερευνητές της συμπερίληψης (1ο Νηπιαγωγείο Προσοτσάνης "Αρετή Βογιατζή Μελαδίνη")




ΠΡΟΓΡΑΜΜΑΤΙΣΜΟΣ ΤΟΥ ΕΡΓΟΥ

Το έργο έχει ως στόχο την εξοικείωση των μαθητών με τις ανοιχτές τεχνολογίες και την αξιοποίησή τους για την εξεύρεση λύσεων που βελτιώνουν την ποιότητα ζωής ατόμων με αναπηρία, προάγουν την συμπερίληψη και προσφέρουν ίσες ευκαιρίες μάθησης σε όλους.
Συγκεκριμένα, η ομάδα “... “ θα σχεδιάσει σταθμούς αισθητηριακής ανακάλυψης για μαθητές που αντιμετωπίζουν αισθητηριακές, μαθησιακές ή άλλες δυσκολίες προκειμένου να τους βοηθήσει να ενταχθούν στην ομάδα του νηπιαγωγείου ισότιμα και να κατακτήσουν τη γνώση σύμφωνα με τον δικό τους ρυθμό.
Για την επίτευξη του στόχου θα χρησιμοποιηθεί το ρομπότ Tale-bot και η πλακέτα MaKey MaKey.


ΕΠΙΜΕΡΟΥΣ ΣΤΟΧΟΙ:

Καλλιέργεια ήπιων και ψηφιακών δεξιοτήτων (5Cs, ψηφιακός εγγραμματισμός, problem solving κτλ)

Ανάπτυξη ενσυναίσθησης και κοινωνικής υπευθυνότητας

Εξεύρεση λύσεων που θα μειώσουν τις ανισότητες και θα ενισχύσουν τη συμπερίληψη

Ευαισθητοποίηση σχετικά με τη δυνατότητα αξιοποίησης της Τεχνολογίας για κοινωνικούς σκοπούς

Εισαγωγή των μαθητών σε έννοιες και διαδικασίες που συνδέονται με τη Ρομποτική, τον Προγραμματισμό, την Α.Ι.

Καλλιέργεια υπολογιστικής σκέψης

Καλλιέργεια ικανότητας συνεργασίας και αποτελεσματικής επικοινωνίας
